# ZJSSO 统一登录门户 - 产品需求文档

## 1. 产品概述

ZJSSO 统一登录门户是面向用户的 OIDC 单点登录前端界面，提供登录注册、OAuth 授权确认、个人信息管理和管理员后台等功能。
- 目标用户：所有需要接入 ZJSSO SSO 系统的终端用户、开发者和管理员
- 核心问题：为 SSO 后端提供可视化的用户交互界面，让用户完成登录、授权确认、个人资料管理等操作

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|----------|
| 普通用户 | 邮箱注册 | 登录、管理个人资料、授权 OIDC 客户端 |
| 开发者 | 邮箱注册 | 包含用户权限 + 注册管理自己的 OIDC 客户端 |
| 管理员 | 系统创建 | 包含开发者权限 + 管理所有用户和客户端 |

### 2.2 功能模块

1. **登录页面**：用户登录、注册、找回密码、第三方登录入口
2. **授权确认页**：OIDC 授权码流程中的用户确认授权界面
3. **OAuth 回调页**：授权回调处理与跳转
4. **用户信息页**：个人资料查看与编辑、WebAuthn 凭证管理、社交账号绑定管理
5. **管理后台**：客户端管理（CRUD）、用户管理（Admin 专用）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 登录页 | 登录表单 | 用户名/邮箱 + 密码登录，极验行为验证 |
| 登录页 | 注册表单 | 用户名、邮箱、密码注册，含极验验证 |
| 登录页 | 找回密码 | 通过邮箱验证码重置密码 |
| 登录页 | 第三方登录 | GitHub、QQ 登录入口 |
| 授权确认页 | 授权信息 | 显示客户端信息、请求的权限范围、用户确认/拒绝 |
| 回调页 | 回调处理 | 接收授权码并重定向 |
| 用户信息页 | 个人资料 | 显示和编辑用户名、邮箱、头像等 |
| 用户信息页 | WebAuthn 凭证 | 注册/查看/删除指纹或硬件密钥 |
| 用户信息页 | 社交绑定 | 查看和绑定/解绑第三方账号 |
| 管理后台 | 客户端管理 | 注册、查看、修改、删除 OIDC 客户端 |
| 管理后台 | 用户管理 | 查看用户列表、创建/编辑/禁用用户（Admin） |

## 3. 核心流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  访问应用   │────►│  检测未登录  │────►│  跳转SSO  │────►│  登录页   │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
                                                       │
                                                    ┌──▼──┐
                                                    │ 登录  │
                                                    └──┬──┘
                                                       │
                                               ┌───────▼───────┐
                                               │   授权确认页    │
                                               │ (需要用户授权时) │
                                               └───────┬───────┘
                                                       │
                                               ┌───────▼───────┐
                                               │ 重定向回客户端   │
                                               │ 携带授权码      │
                                               └───────────────┘
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝 #0f172a（背景）、靛蓝 #3b82f6（主色）、白色 #ffffff（文字）
- **辅色调**：翠绿 #10b981（成功）、橙红 #ef4444（错误/删除）
- **按钮风格**：圆角 8px，主按钮渐变背景，hover 微动效
- **字体**：系统无衬线字体栈
- **布局风格**：居中卡片式布局，深色主题，玻璃态效果（毛玻璃卡片）
- **图标风格**：Font Awesome 6 免费图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 登录页 | 登录表单 | 居中毛玻璃卡片、输入框组、登录按钮、极验容器、第三方登录按钮组 |
| 登录页 | 注册/找回密码 | 选项卡切换、验证码输入、密码强度指示 |
| 授权确认页 | 授权卡片 | 应用图标和名称、权限列表、确认/拒绝按钮 |
| 用户信息页 | 资料卡片 | 头像、字段列表、编辑按钮、设置卡片网格 |
| 管理后台 | 数据表格 | 侧边导航、搜索栏、数据表格、编辑弹窗 |

### 4.3 响应式设计

- 桌面优先，登录页和授权页在移动端适配为全屏布局
- 管理后台桌面端专用（最小宽度 1024px）

## 5. 非功能需求

- 支持现代浏览器最新两个版本
- 页面加载时间不超过 2 秒
- Token 过期前 5 分钟自动尝试刷新
